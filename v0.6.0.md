## Measurement Model (Real-world)

START:
Mått tas från kopplingens öppning.
Instick måste därför adderas.

```
adj[0] = -build + insertionDepth
```

MIDDLE:
Oförändrad.

END (CC):
Mått tas till kopplingens centrum.

```
adj[last] = -La
```

Regler:
- START använder `insertionDepth`, inte `La`
- END (CC) använder `La`, inte `insertionDepth`
- `insertionDepth` används endast på START
- `La` används endast på END (CC)
- Ingen blandning av `insertionDepth` och `La` på samma sträcka

## Justeringslogik

**Start:**
```
adj[0] = -build + insertionDepth
```

**Mellan:**
```
adj[i] = oförändrad
```

**Slut — ej CC:**
```
adj[n−1] = +build + La_slut
```

**Slut — CC (ersätter hela formeln):**
```
adj[n−1] = −La_slut
```

Build appliceras aldrig på slutände när CC är aktivt.

## UI (inline i sträckraden)

```
Sträcka 1  [ 232 ]  [VA 1130 Skarv ▼]  [ ← | → ]  [×]
Sträcka 2  [ 405 ]                     [ ← | → ]  [×]
Sträcka 3  [ 133 ]  [VA 1150 T-stycke ▼] [☑ CC]   [×]
```

- Kopplingsval visas på start-rad (rad 1) och slut-rad (sista raden)
- Mellansträckor har ingen kopplingskolumn
- Väljer användaren koppling → La uppdateras automatiskt från fittings.json
- Väljer användaren "— ingen —" → La = 0, ingen justering
- `defaultCC` i fittings.json sätter CC-checkboxen automatiskt
- CC-checkbox visas alltid på slutraden, oavsett läge
- La-värde hanteras internt (ej synligt fält)

## Dataflöde

```
fittings.json  →  mbFittings[]  (fetch, window.onload)
      │
mbFittingSelected('start')
      └─ insertionDepth  ← insertionDepth[Dy]
      │
mbFittingSelected('slut')
      ├─ La  ← dimensioner[Dy].La
      └─ CC  ← defaultCC
      │
mbRecomputeAdjustments()
      └─ start: −build + insertionDepth
      └─ slut CC: −La
```

## Referensfall

**Scenario:** Skarvkoppling (start) → T-stycke CC (slut)

| Sträcka | Bas | Typ | Justering | Effektiv |
|---------|-----|-----|-----------|---------|
| S1 | 232 | start, öppning → vägg | −build + instick | 232 − 20 + 10 = 222 |
| S2 | 405 | mellansträcka | −build | 405 − 20 = 385 |
| S3 | 133 | slut, vägg → CC | −La_slut | 133 − 12 = 121 |

build = 20 mm, insertionDepth_start = 10 mm (VA 1130 / 12mm), La_slut = 12 mm (VA 1150 / 12mm)
